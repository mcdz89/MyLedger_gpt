# MyLedger_gpt

Notice: This application was generated in part with the help of an AI LLM.

MyLedger is a personal finance desktop app for Ubuntu (Python 3.11 + GTK4/libadwaita + SQLite 3) focused on privacy and simplicity.

## Run

```bash
python3 -m app.main
```

Optional `.env` override:

```env
DATABASE_PATH=db/myledger.sqlite3
```
